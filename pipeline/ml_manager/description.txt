Как они работают вместе

Обучение:
modelTrainerManager инициирует обучение новой модели, задает гиперпараметры и передает данные модели.

После обучения, модель может быть зарегистрирована с помощью modelRegistryManager.

Оценка: Обученная модель передается в modelEvaluatorManager для оценки. Метрики производительности могут быть сохранены или логированы.

Регистрация и хранение: После успешной оценки модель регистрируется и сохраняется с помощью modelRegistryManager.

Развёртывание: Если модель одобрена для продакшн, modelDeployerManager разворачивает модель в нужной среде.
В случае проблем можно использовать откат к предыдущей версии с помощью modelRegistryManager.


Пример процесса использования:

Модель обучается с помощью modelTrainerManager.
Оценка проводится через modelEvaluatorManager.
Модель регистрируется в modelRegistryManager.
Если требуется, модель разворачивается через modelDeployerManager.
